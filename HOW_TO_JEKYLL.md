1. Télécharger Ruby v3.2.3 (x64) sur https://rubyinstaller.org/downloads/archives/

2. Lancer l'installation, puis, dans la fenêtre qui s'ouvre à la fin de l'installation, entrer 3.
    
3. Dans un terminal, 
    1. lancer `gem install bundler` 
    2. changer de répertoire avec `cd C:/Users/flori/Desktop/site-web-balan`
    3. lancer `bundle install`
    3. lancer `bundle update github-pages`

4. Lancer `bundle exec jekyll serve --livereload` pour mettre à jour le rendu du site local
